Anchored reactions — design variants

Select text in any document → floating toolbar → 😊 → pick from the curated set (👍 👎 🎉 ❤️ 😄 🚀 👀). The reaction is visually tied to that span. Each variant explores a fundamentally different home for the marginalia. All share one mock model ({id, emoji, author, anchor, comment_id}), the chosen variant-B comment rail, and real Gravatars for raf@kernel.sh / hi@raf.xyz.

Mock data deliberately includes: a span reacted by multiple authors with stacked emoji; a span carrying both a comment highlight and reactions (the hard overlap case — the "either a write landed whole…" sentence); doc-level reactions; and a reaction on the code block. Click your own reaction to toggle it off. Comment highlight = solid yellow; reaction anchor = quiet blue — two distinct languages.
